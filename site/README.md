# PlayToys Clone Example

Este directorio contiene un ejemplo muy sencillo de un portal de videos al estilo de playtoys.tv. Los archivos principales son:

- `index.html`: pagina de inicio con una galeria de videos.
- `video1.html` y `video2.html`: paginas de ejemplo que reproducen un video mediante la etiqueta HTML5 `<video>`.
- `style.css`: estilos basicos para la pagina.
- `script.js`: script opcional de ejemplo.

Para ver el sitio, abre `index.html` en tu navegador. Puedes alojar estos archivos en cualquier servidor estatico o en GitHub Pages.
